# Books_data_scraping

This project scrapes book data from a website and saves the results in both CSV and JSON formats. It includes a Python script for scraping, and the output files contain information about books from multiple pages.

## Files

- **Bookdata.py**: The main Python script for scraping book data.
- **books_all_50_pages.csv**: The CSV file containing the scraped book data.
- **books_all_50_pages.json**: The JSON file containing the scraped book data.

## Usage

1. Make sure you have Python installed.
2. Run `Bookdata.py` to start scraping and generate the output files.
3. The results will be saved in both CSV and JSON formats in the same directory.
